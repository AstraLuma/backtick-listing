FAQ

Why isn't my command listed?

An attempt is made to filter out "bad" commands. There are several reasons this happens:

  1. You didn't actually write a command, leaving the default behavior in place.
  2. You didn't name your command, leaving it unfindable

To fix this, please go edit your gist. When these problems are resolved, this page will reflect it.
